Listings press pack

Everything a directory form asks for, in one place, worded once and reused everywhere. Three properties: Greenwood Upholstery (local listings), Learn to Upholster (craft & resource listings) and Visualiser Pro (AI tool directories). Copy — don't retype.

How to use this with Claude in Chrome

  1. Keep this page open in one tab and the directory-tracker spreadsheet handy.
  2. Pick the next directory in the tracker and open its start URL in a new tab.
  3. Tell Claude: "Fill in this listing form using my press pack in the other tab — this one's for Greenwood Upholstery" (or Learn to Upholster / Visualiser Pro).
  4. You click the captcha and the final submit button — Claude stops and asks before anything is sent. Type any passwords yourself.
  5. Mark the tracker: status, date, login email used — and paste the live link once the listing is approved.

Exact NAP, everywhere. Name, address and phone must match this page to the letter on every listing. Mixed details confuse Google and dilute the value.

One login email. Pick a single address for all directory accounts (pat@greenwoodupholstery.com, or set up a dedicated listings@ alias) and record it in the tracker each time.

Passwords live in your password manager. Never in the tracker, never on this page, never pasted to Claude.

Pace it. Three to five complete, photo-rich submissions a day beats fifty thin ones — and looks natural.

Free tiers only, unless the tracker says a paid one is genuinely worth it.

Fix these while you're at it: Yorkshire.com currently shows your opening hours as Mon–Thu 9–3, Fri 9–2:30 — your own site says Mon–Fri 7:00–16:00. Claim and correct it. A few older listings also give the address as "Mytholmroyd" — standardise on the address block below.
